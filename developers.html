Developers

Ratemarkt is built by developers again for the use of developers to make it quite easy to get connected. Please choose your integration type and start getting integrated with us!


Become a Client

If you'd like to get integrated with Ratemarkt as a client, please follow our API documentation. Once you're done dont' forget to certificate your integration using our online integration testing tool. With Ratemarkt's simple and intuitive API design you'll be done in a couple of days!

Get Started!

Become a Supplier

Do you want to take place as a supplier in Ratemarkt's suppliers collection? Then you might consider to build an API service which conforms our outbound API connector specification. Once you are done, Ratemarkt can automatically connect to your API and you can start selling instantly!

Get Started!

Extend Ratemarkt

Enhance Ratemarkt functionality for your custom needs if you like to! Just follow our SDK documentation and start developing a Ratemarkt Plugin which can easily get plugged into your API flow. Once you're done with your plugin our technical team will review your implementation and install the plugin into the system for your own use.

Get Started!
